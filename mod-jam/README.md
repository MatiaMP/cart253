## Description

> *This Mod Jam is a game where the user controls a frog on a diet in which the goal of the game is to eat vegetables and not the junk food. In this case it is broccoli vs burgers.

> When eating broccoli, the score will go up +1, when eating burgers the goal will go down -1

> If the score reaches +5, you win, if the score reaches -5, you lose

> You can move the frog with the mouse or arrow keys

> You can launch the tongue with left click or spacebar
